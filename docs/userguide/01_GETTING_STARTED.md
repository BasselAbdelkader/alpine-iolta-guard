## Welcome to IOLTA Guard!

IOLTA Guard is your trust accounting management system. This guide will help you get started quickly and easily - no technical knowledge required!

### What is IOLTA Guard?

IOLTA Guard helps your law firm manage client trust accounts. Think of it as a specialized checkbook for client funds that also:
- Tracks every deposit and withdrawal
- Organizes funds by client and case
- Generates reports for compliance
- Keeps a complete audit trail

### Who Can Use This System?

- **Attorneys:** Full access to manage clients, cases, and funds
- **Paralegals:** Add clients, cases, and transactions
- **Bookkeepers:** Manage financial transactions and reconciliation
- **Office Administrators:** Full system access

## Logging In

### Step 1: Open Your Web Browser

- Open Chrome, Firefox, Safari, or Edge
- Type your IOLTA Guard web address in the address bar
- Example: `http://your-firm-url.com` or `http://192.168.1.100`

**Screenshot placeholder:** [URL_HERE - Login Page]

### Step 2: Enter Your Credentials

You'll see a login screen with two boxes:

1. **Username:** Type the username given to you by your administrator
2. **Password:** Type your password (it will appear as dots for security)

Click the blue "Login" button. You'll be taken to the Dashboard.

## Your First Tasks

Let's walk through the most common tasks you'll do in IOLTA Guard.

### Task 1: Add a New Client

**Why?** Before you can manage money, you need to add the client.

**Steps:**

1. Click "**Clients**" in the menu
2. Click the "**+ Add New Client**" button (usually top-right)
3. Fill in the client information:
   - First Name (required)
   - Last Name (required)
   - Email (optional, but recommended)
   - Phone (optional, but recommended)
   - Address, City, State, Zip (optional)
4. Click "**Save**"

**Screenshot placeholder:** [URL_HERE - Add New Client Form]

**What Happens Next:**
- Your new client appears in the client list
- You can now create cases for this client
- You'll receive a success message: "Client created successfully!"

### Task 2: Create a Case for the Client

**Why?** Each legal matter (case) needs its own record to track funds.

**Steps:**

1. Go to **Clients** menu
2. Find your client in the list (use search if needed)
3. Click on the client's name
4. You'll see the client detail page
5. Click "**+ Add New Case**" button
6. Fill in case information:
   - Case Number (required) - Example: "2025-001"
   - Case Title (required) - Example: "Personal Injury Settlement"
   - Matter Type (optional) - Example: "Personal Injury"
   - Opened Date (required) - Date case started
   - Description (optional) - Brief description of case
7. Click "**Save**"

**Screenshot placeholder:** [URL_HERE - Add New Case Form]

**What Happens Next:**
- Case is created and linked to the client
- Case appears in the client's case list
- You can now record transactions for this case

### Task 3: Record a Deposit

**Why?** When you receive money for a client (settlement, retainer, etc.)

**Steps:**

1. Go to the client's page (Clients → Click client name)
2. Find the case in the case list
3. Click "**Add Transaction**" or navigate to the case detail page
4. Select transaction type: "**Deposit**"
5. Fill in deposit information:
   - Amount (required) - Example: $10,000.00
   - Transaction Date (required)
   - Bank Account (required) - Select the trust account
   - Description (optional but recommended) - Example: "Settlement payment received"
6. Click "**Save**"

**Screenshot placeholder:** [URL_HERE - Add Deposit Form]

**What Happens Next:**
- Money is added to the case balance
- Transaction appears in the case transaction list
- Client's total balance increases

### Task 4: Record a Withdrawal

**Why?** When you pay money out from the trust account

**Steps:**

1. Go to the case detail page
2. Click "**Add Transaction**"
3. Select transaction type: "**Withdrawal**"
4. Fill in withdrawal information:
   - Amount (required) - Example: $500.00
   - Transaction Date (required)
   - Bank Account (required)
   - Payee (required) - Who received the money
   - Description (optional but recommended) - Example: "Medical records fee"
   - Check Number (if applicable)
6. Click "**Save**"

**Screenshot placeholder:** [URL_HERE - Add Withdrawal Form]

**What Happens Next:**
- Money is subtracted from the case balance
- Transaction appears in the transaction list
- Client's total balance decreases

**Important:** The system will prevent you from withdrawing more money than available!

## Getting Help

### In-App Help
- Look for the **?** icon or "Help" link
- Tooltips appear when you hover over fields

### Common Questions

**Q: Can I delete a transaction?**
A: No. For audit purposes, transactions cannot be deleted. If you made a mistake, you can "void" the transaction. Contact your supervisor.

**Q: What if I enter the wrong amount?**
A: You cannot edit transactions. You'll need to void the incorrect transaction and create a new correct one. Contact your supervisor for help.

**Q: How do I know if funds are available before withdrawal?**
A: The system shows the current balance for each case. The system will also prevent withdrawals if insufficient funds.

**Q: Can I see all transactions for a client?**
A: Yes! Go to the client detail page, and you'll see all cases and transactions.

**Q: How do I generate reports?**
A: See the [Reports Guide](./04_REPORTS_GUIDE.md) for detailed instructions.
