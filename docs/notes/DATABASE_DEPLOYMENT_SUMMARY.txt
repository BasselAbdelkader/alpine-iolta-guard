MANUAL DEPLOYMENT (ALTERNATIVE)
================================================================================

If you prefer manual control:

$ cd ve_demo

$ docker-compose build --no-cache
$ docker-compose up -d
$ sleep 15
$ docker exec iolta_db psql -U iolta_user -d postgres -c "DROP DATABASE IF EXISTS iolta_guard_db; CREATE DATABASE iolta_guard_db;"
$ cat backups/iolta_production_dump.sql | docker exec -i iolta_db psql -U iolta_user -d iolta_guard_db
